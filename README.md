# grace
Grace, the bot

# Install and usage

```bash
$ cp .aws-env-sample .aws-env
$ nano .aws-env  # Add your own account info
$ source .aws-env
$ apex deploy  # deploy functions
$ apex bots  # list bots
$ apex upbot  # create/update gracebot
$ apex hello  # say hello
```
